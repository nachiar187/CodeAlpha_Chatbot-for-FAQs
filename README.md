# Chatbot for FAQs

## Description
This project is a FAQ Chatbot developed using python. It answers common questions by matching user queries with predefined FAQs.

## Features
- Answers frequently asked questions
- Simple and user-friendly interface
- Uses text similarity to find the best answer
- Built with Python and Tkinter

## Technologies Used
- Python
- Tkinter
- Scikit-learn

## How to Run
1. Install the required libraries:
   pip install scikit-learn
2. Run the Python file:
   py faq_chatbot.py
